# Computer
Давай соберем компьютер. Вот что тебе нужно сделать:

Создай интерфейс CompItem.
Добавь в него метод String getName().
Создай классы Keyboard, Mouse, Monitor, которые реализуют интерфейс CompItem.
Метод getName() должен возвращать имя класса, например, для класса Keyboard будет "Keyboard".
Создай класс Computer.
В класс Computer добавь приватное поле keyboard типа Keyboard.
В класс Computer добавь приватное поле mouse типа Mouse.
В класс Computer добавь приватное поле monitor типа Monitor.
Создай конструктор с тремя параметрами в классе Computer используя комбинацию клавиш Alt+Insert (для Windows) внутри класса (команда Constructor).
Внутри конструктора инициализируйте все три поля (переменных) класса в соответствии с переданными параметрами.
Создай геттеры для полей класса Computer (в классе используй комбинацию клавиш Alt+Insert (для Windows) и выбери команду Getter).
Все созданные классы и интерфейс должны быть в отдельных файлах.


Псевдокод:

```
АЛГОРИТМ ComputerComponents

    ИНТЕРФЕЙС КомпьютерныйКомпонент
        ФУНКЦИЯ ПолучитьИмя() : Строка
    КОНЕЦ ИНТЕРФЕЙСА

    КЛАСС Клавиатура РЕАЛИЗУЕТ КомпьютерныйКомпонент
        ФУНКЦИЯ ПолучитьИмя() : Строка
            ВОЗВРАТ "Keyboard"
        КОНЕЦ ФУНКЦИИ
    КОНЕЦ КЛАССА

    КЛАСС Мышь РЕАЛИЗУЕТ КомпьютерныйКомпонент
        ФУНКЦИЯ ПолучитьИмя() : Строка
            ВОЗВРАТ "Mouse"
        КОНЕЦ ФУНКЦИИ
    КОНЕЦ КЛАССА

    КЛАСС Монитор РЕАЛИЗУЕТ КомпьютерныйКомпонент
        ФУНКЦИЯ ПолучитьИмя() : Строка
            ВОЗВРАТ "Monitor"
        КОНЕЦ ФУНКЦИИ
    КОНЕЦ КЛАССА

    КЛАСС Компьютер
        СВОЙСТВА:
            клавиатура: Клавиатура
            мышь: Мышь
            монитор: Монитор

        КОНСТРУКТОР(клавиатура: Клавиатура, мышь: Мышь, монитор: Монитор)
            ЭТО.клавиатура = клавиатура
            ЭТО.мышь = мышь
            ЭТО.монитор = монитор
        КОНЕЦ КОНСТРУКТОРА

        ФУНКЦИЯ ПолучитьКлавиатуру() : Клавиатура
            ВОЗВРАТ клавиатура
        КОНЕЦ ФУНКЦИИ

        ФУНКЦИЯ ПолучитьМышь() : Мышь
            ВОЗВРАТ мышь
        КОНЕЦ ФУНКЦИИ

        ФУНКЦИЯ ПолучитьМонитор() : Монитор
            ВОЗВРАТ монитор
        КОНЕЦ ФУНКЦИИ
    КОНЕЦ КЛАССА

    ГЛАВНЫЙ АЛГОРИТМ
        // Создание компьютера с компонентами
        компьютер = НОВЫЙ Компьютер(НОВАЯ Клавиатура(), НОВАЯ Мышь(), НОВЫЙ Монитор())
        
        // Проверка работоспособности
        ЕСЛИ ПроверитьРаботу(компьютер.ПолучитьКлавиатуру()) И
           ПроверитьРаботу(компьютер.ПолучитьМонитор()) И
           ПроверитьРаботу(компьютер.ПолучитьМышь()) ТО
            ВЫВЕСТИ "Work!"
        КОНЕЦ ЕСЛИ
    КОНЕЦ ГЛАВНОГО АЛГОРИТМА

    ФУНКЦИЯ ПроверитьРаботу(компонент: КомпьютерныйКомпонент) : Логическое
        имя = компонент.ПолучитьИмя()
        ВЫВЕСТИ имя
        ВОЗВРАТ (имя НЕ РАВНО null) И (ДЛИНА(имя) > 4)
    КОНЕЦ ФУНКЦИИ

КОНЕЦ АЛГОРИТМА
```
